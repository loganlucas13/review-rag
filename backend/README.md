# Backend Setup

> [!IMPORTANT]
> Make sure to have a Python3 version >= 3.9 installed before continuing.

1. Activate Python virtual environment by running the following commands from the backend folder
    - Mac/Linux: `python3 -m venv .venv`, then `. .venv/bin/activate`
    - Windows: `py -3 -m venv .venv`, then `.venv\Scripts\activate`
2. Install flask by running `pip install Flask` in the terminal.
